# Projekt funkciók

## 1. Kérések megtekintése
- **Guest login:** csak megtekintés, nem lehet jelentkezni
- **User login / Google integráció:**
  - kérések megtekintése
  - jelentkezés
  - elfogadás

## 2. Profiloldal
- Személyes adatok: megtekintés és módosítás
- Kijelentkezés
- Beállítások

## 3. Posztok
### Létrehozás
- Cím, leírás, téma
- Pénzjutalom
- Dokumentumok feltöltése

### Elfogadás / Lezárás
- Elfogadás → feladat az elfogadónak
- Lezárás kölcsönösen történik

### Törlés
- User, moderátor és admin is törölheti

### Poszt állapotok
- Új
- Elvállalva
- Kész

### Keresés
- Posztok / település alapján

## 4. Privát chat
- Tartalmazza a létrehozót és az elfogadót

## 5. Kommentek (nyilvános)
- Bárki kommentelhet a poszt alá

## 6. User statok / Dashboard
- Összes kérés
- Létrehozott kérések száma
- Teljesített kérések száma

## 7. Szerepkörök
- User
- Moderátor
- Admin

## 8. Értékelés
- Igen/nem válaszok, részletes kifejtéssel
